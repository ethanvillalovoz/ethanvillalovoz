# Ethan Villalovoz

I build robot-learning and agentic systems at [Georgia Tech](https://www.gatech.edu/) and [Microsoft](https://www.microsoft.com/). I care about how intelligent systems learn from feedback, behave under uncertainty, and get evaluated.

At Georgia Tech, I am working on video-to-action robot policy learning. At Microsoft, I am building multi-agent workflow integrations for an internal cloud platform.

## Selected Work

- [ScenarioLens](https://ethanvillalovoz.com/scenariolens/) ranks and inspects long-tail driving scenarios with interpretable interaction, risk, and baseline-failure metrics.
- [MetricDrive](https://ethanvillalovoz.com/metricdrive/) studies metric-aligned trajectory choices, preference pairs, hard negatives, and VLM-style planning rows.
- [BODE-GEN](https://arxiv.org/abs/2512.15076) applies Bayesian prompt optimization to test-driven code generation across HumanEval+ tasks and multiple code models.
- [Clarification-Guided Reward Learning](https://github.com/ethanvillalovoz/clarification-guided-reward-learning) investigates state correction and feature clarification for ambiguous robot preferences.
- [IntelliCrawl](https://github.com/ethanvillalovoz/intellicrawl) produces source-backed developer-tool comparisons through a validated LangGraph research pipeline.
